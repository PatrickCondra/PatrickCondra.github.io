Global Route and Task Generalization

This project was assigned as part of my senior capstone design course at Texas A&M, and the aim of the project was to implement a system which allowed for users to modify vehicle behavior triggers for autonomous vehicles through a map GUI.

Background

In our project, we were provided with a set of HDMap data files, and rosbag containers, which contained data that would simulate the location of a car driving in realtime.
